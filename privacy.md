# Panda Fun Bot – Privacy Policy

Last Updated: 17-02-2026

Panda Fun Bot (“Panda Fun”, “the Bot”, “we”, “our”, “us”) respects your privacy. This Privacy Policy explains what information we collect, how we use it, and your rights regarding that information.

By using Panda Fun Bot, you agree to this Privacy Policy.

---

## 1. Information We Collect

Panda Fun Bot only collects data necessary to operate its features.

### A. Discord User Data

* Discord User ID
* Username and discriminator
* Guild (Server) ID
* Channel ID
* Role IDs (for permission checks)

We do NOT collect:

* Passwords
* Email addresses
* Private Discord messages (unless part of a game interaction)
* IP addresses
* Billing or payment information

---

### B. Game & Economy Data

To provide gameplay features, we store:

* Coin balance
* XP and level
* PvP statistics
* Daily streak data
* Inventory items
* Match history (if applicable)
* Guild-specific settings (prefix, feature toggles)

---

### C. Temporary Session Data

Some features (e.g., Blackjack, Mines, PvP matches) store temporary in-memory session data.
This data is automatically cleared when:

* The game ends
* The bot restarts
* The session expires

---

## 2. How We Use Your Data

We use stored data only to:

* Operate game features
* Maintain economy balances
* Track progression (XP, levels, PvP stats)
* Provide leaderboards
* Enforce cooldowns and anti-abuse measures
* Improve system stability

We do NOT:

* Sell data
* Share data with third parties
* Use data for advertising
* Profile users for marketing

---

## 3. Data Storage & Security

* Data is stored securely using database systems (e.g., Prisma + database backend).
* Only the bot owner/operator has administrative access.
* We take reasonable measures to prevent unauthorized access.

However, no system is 100% secure.

---

## 4. Data Retention

We retain data for as long as:

* The bot remains in a server
* The user continues to use the bot

Data may be deleted if:

* The bot is removed from a server
* A user requests deletion (see Section 6)
* The database is reset for maintenance

We are not responsible for Discord’s data practices.
